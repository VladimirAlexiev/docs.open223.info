Open223 Resources#

The Open223 GitHub organization was created in conjunction with the ASHRAE Standard 223 project committee to develop resources to support the use of Standard. However, it is not an official ASHRAE product or a part of the standard.

Open223 Models#

This site contains the canonical sources for example models.

Open223 Explore#

This page hosts autogenerated documentation of the 223P ontology, emphasizing the availability and correct use of concepts and properties defined by the ontology.
